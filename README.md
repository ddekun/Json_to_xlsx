# Json_to_xlsx
Проект на Python для обработки и подготовки JSON-данных к аналитике премий по интенсивности.

В проекте загружается исходный JSON-файл с данными по сотрудникам, из вложенной структуры извлекаются отдельные JSON-блоки и разворачиваются в плоскую табличную модель. Для каждой записи сотрудника дополнительно подтягиваются общие атрибуты блока, включая `totalAmount` и `premiumType`, что позволяет анализировать как индивидуальные начисления, так и общий объём премий по каждому виду.

Результат обработки собирается в `pandas DataFrame`, выгружается в Excel и используется для последующего анализа количества сотрудников, сумм начислений и структуры премий по интенсивности.
